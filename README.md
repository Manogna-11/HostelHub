# 🏨 HostelHub AI

### AI-Powered Hostel Discovery, Booking & Management Platform

HostelHub AI is a modern full-stack web application that simplifies hostel discovery, booking, and management for students and hostel administrators. The platform combines intelligent search, complaint management, analytics, hostel administration, and AI-powered automation into a single solution.

Designed for educational institutions, private hostels, and student accommodations, HostelHub AI provides a seamless experience for both hostel seekers and administrators.

## 📋 Project Overview

HostelHub AI digitizes hostel operations and student accommodation services.

### 👨‍🎓 For Students

Students can:

* Search hostels by location and budget
* View hostel facilities and room details
* Submit maintenance complaints
* Track complaint status
* View hostel notices
* Access weekly mess menus
* Submit feedback and ratings
* Interact with an AI assistant

### 🛡️ For Administrators

Administrators can:

* Manage hostel rooms
* Assign students to rooms
* Manage complaints
* Publish notices
* Update mess schedules
* Review student feedback
* Monitor occupancy statistics
* Analyze hostel performance through dashboards

## 🤖 AI Features

### 1. Complaint Classification

The AI system automatically:

* Detects complaint category
* Assigns priority level
* Generates maintenance summaries

Examples:

* Electrical Issue
* Plumbing Issue
* Internet Issue
* Maintenance Issue

### 2. HostelHub Assistant

An intelligent chatbot capable of answering:

* Hostel Rules
* Room Allocation Procedures
* Complaint Processes
* Mess Timings
* Hostel Facilities
* Navigation Support

Features:

* Chat History
* Suggested Questions
* Typing Animation
* Context-Aware Responses

### 3. Notice Summarization

The AI automatically generates:

* Notice Summary
* Key Points
* Important Deadlines
* Action Items

### 4. Feedback Sentiment Analysis

The system analyzes student feedback and provides:

* Positive / Neutral / Negative Classification
* Sentiment Score
* AI Summary

## 🗄️ Database Schema

| Table        | Purpose                                      |
| ------------ | -------------------------------------------- |
| profiles     | Student and administrator information        |
| user_roles   | Role-based access control                    |
| rooms        | Room details, occupancy, capacity and status |
| complaints   | Complaint information and AI summaries       |
| notices      | Hostel announcements and AI summaries        |
| feedback     | Ratings, sentiment scores and AI insights    |
| mess_menu    | Daily meal schedules                         |
| chat_history | AI assistant conversation history            |
