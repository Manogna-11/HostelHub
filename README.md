## Description:
HostelHub is a simple web application that helps students discover and book hostels across multiple cities. 
Users can browse hostels, filter them based on facilities and price, view detailed hostel information, save their favorite hostels, and submit booking requests.
The project is built using HTML, CSS, and JavaScript and uses LocalStorage to store saved hostels and booking details.


##  Features
- Browse hostels from multiple cities
- Filter hostels by price and facilities
- View detailed hostel information
- Save favorite hostels
- Submit hostel booking requests
- Interactive hostel details page
- Local storage for saved data
